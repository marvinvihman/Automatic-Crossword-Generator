Automatic Crossword Generator

My bachelor's thesis.

The program's algorithm for making crosswords is a little funky right now, for some reason it sometimes overwrites words, that have already been added to the table. Current problems also include some words missing definitions in the Estonian Wordnet.

This project can only run on Python versions 3.7-3.9, because EstNLTK supports only those currently.

To run the program locally, it needs to have EstNLTK, Flask and Pandas installed. For easier use, it's recommended to download and install Anaconda, and for less disk usage Miniconda can also be installed.

Using Anaconda or Miniconda GUI version or the Anaconda (Powershell) Prompt, do the next steps.

conda create -n env_name python=3.9

conda activate env_name

Now when virtual env has been created, then you can install the needed moduls.

conda install -c estnltk -c conda-forge estnltk

pip install flask

pip install pandas

This project has many ways, that it can be improved upon, and those will be kept for the future.

Automaatne ristsõna genereerija

Tegu on minu bakalaureuse tööga.

Programmi algoritm hetkel ei tööta korrektselt ning selles on vead sees, vahest kirjutades üle lauale juba lisatud sõnasid. Lisaks on Estonian Wordnet'ist mõnede sõnade definitsioonid puudu, mille tõttu hetkel läheb vihjeks sõna ise.

See projekt töötab hetkeseisuga ainult Python versioonide 3.7-3.9'ga, sest EstNLTK toetab ainult neid, mitte uuemat.

Käivitada programmi lokaalselt oma arvutis on vaja eraldi mooduleid alla laadida, milleks on EstNLTK, Flask ja Pandas. Lihtsamaks kasutamiseks on soovituslik alla laadida Anaconda, vähesema ruumi kasutamiseks on võimalik ka Miniconda alla laadida.

Kasutades Anaconda või Miniconda GUI'd, sobib ka Anaconda (Powershell) Prompt, tee järgmised sammud.

conda create -n env_name python=3.9

conda activate env_name

Now when virtual env has been created, then you can install the needed moduls.

conda install -c estnltk -c conda-forge estnltk

pip install flask

pip install pandas

Projektil on mitmeid viise, kuidas seda arendada saaks, kuid see on tulevikuks.
